## __gcd()

- __gcd() 可以求得两个数的最大公约数

```C++
#include<iostream>
#include<algorithm>
using namespace std;
int main()
{
	int a,b;
	cin>>a>>b;
	cout<<__gcd(a,b)<<endl;
}
```

- 求最大公约数一般是用辗转相除法
